# Global Watcher

**Automatic local file history for the work you cannot afford to lose.**

Global Watcher is a Windows desktop app that watches the folders you choose and keeps a local history of file changes. If a document is overwritten, a chapter disappears, a project file is damaged, or an AI tool changes the wrong file, Global Watcher helps you look back, compare versions, and restore what you need.

> Preview notice: Global Watcher is currently a preview release. Test it on non-critical folders first and keep independent backups of important work.

## Why It Exists

People lose work in very ordinary ways:

- a file is saved over by mistake;
- an old draft is replaced by a weaker new one;
- a folder is changed by a tool, script, editor, or AI assistant;
- a project breaks before there is time to make a backup;
- Git was not set up yet, or the user never learned Git;
- cloud sync copies the bad version everywhere.

Global Watcher is built for those moments. It gives local files a memory.

## Who It Is For

Global Watcher is designed for people who create, edit, revise, and protect important local work.

- **Writers and poets** who want to recover older lines, drafts, scenes, verses, and notes.
- **Students and researchers** who work with papers, notes, datasets, drafts, and experiments.
- **Developers** who want extra protection before, between, or outside Git commits.
- **Beginners** who need file history without learning version control first.
- **Lawyers, editors, and document-heavy professionals** who cannot afford accidental overwrites.
- **Designers and creators** who work with evolving project folders and assets.
- **AI-assisted workers** who want a safety layer when tools can change many files quickly.

## What It Does

- Watches selected folders on your computer.
- Saves local file versions as changes happen.
- Shows recent activity and file history.
- Lets you browse earlier versions.
- Helps restore files after accidental edits, deletes, or overwrites.
- Includes preview and comparison workflows for supported files.
- Gives you a built-in place to inspect changes without opening a full development environment.
- Works as a companion layer for Git, cloud sync, and normal backups.

## Built-In Preview And Comparison

Global Watcher is not only a background watcher. It also gives you a visual control center where you can review what changed.

For supported files, you can inspect versions, preview content, and compare differences before deciding what to restore. This is useful for code, notes, documents, drafts, and text-heavy work where one small change can matter.

## For People Who Do Not Use Git

Git is powerful, but many people do not use it:

- writers;
- poets;
- students;
- legal workers;
- office workers;
- designers;
- people who simply need their files to be safer.

Global Watcher is made to be understandable without commits, branches, pull requests, or command-line workflows. Choose folders, work normally, and use history when something goes wrong.

## For Developers Who Already Use Git

Global Watcher is not a Git replacement. It is a safety layer around Git.

It can help when:

- you changed files before committing;
- a generator or AI coding tool modified too much;
- a temporary experiment was never saved in Git;
- a file was deleted outside your editor;
- you want to compare what happened locally before deciding what to commit.

Git protects intentional checkpoints. Global Watcher helps cover the messy time between checkpoints.

## Local-First By Design

Global Watcher is built around local work:

- watched folders are chosen by the user;
- history is kept locally on the user's machine;
- no account is required for the core local workflow;
- users remain responsible for their own backup strategy.

## What Global Watcher Is Not

Global Watcher is not a promise that every file can always be recovered.

It is not:

- a replacement for independent backups;
- a replacement for Git;
- a cloud backup service;
- antivirus software;
- a legal archive or forensic evidence system;
- a guarantee against data loss.

Use it as an additional safety layer, not as the only copy of important work.

## Preview Status

Global Watcher is currently a preview release. That means it is being improved and should be tested carefully before being trusted with critical workflows.

Recommended first test:

1. Create a small demo folder.
2. Add a few sample text files.
3. Watch the folder with Global Watcher.
4. Edit and delete test files.
5. Review history and restore behavior.
6. Only then decide how you want to use it on real work.
